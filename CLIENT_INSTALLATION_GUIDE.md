# Client Installation Guide

### Minimum Requirements
- **Operating System**: Windows 10 or later (64-bit)
- **Memory**: 4GB RAM minimum, 8GB recommended
- **Storage**: 500MB available disk space

- **Operating System**: Windows 11 (64-bit)
- **Memory**: 8GB RAM or more

- **Processor**: Intel Core i3 or AMD equivalent

## Installation Steps

### Step 1: Download the Installer
1. Obtain the installer file: `Enterprise Lead Management System Setup 2.0.0.exe`
2. Ensure the file is complete (approximately 150MB)
3. Verify the installer is not corrupted

### Step 2: Run the Installer
1. **Double-click** the installer file
2. **Accept the license agreement** when prompted
3. **Wait** for the installation to complete automatically
4. The installer will:

## Troubleshooting

### Installation Issues

**Problem**: "Windows protected your PC" SmartScreen warning
**Solution**: 
1. Click "More info"
2. Click "Run anyway"
3. This is normal for new applications without code signing

**Problem**: Installation fails or hangs
**Solution**:
1. Ensure you have at least 500MB free disk space
2. Close other applications to free up RAM
3. Temporarily disable antivirus software
4. Run the installer as administrator (right-click → Run as administrator)

**Problem**: Application won't start after installation
**Solution**:
1. Check system requirements (Windows 10+, 4GB RAM)
2. Try running as administrator
3. Check for antivirus blocking the application
4. Reinstall the application

### Performance Issues

**Problem**: Application runs slowly
**Solution**:
1. Close other applications to free up RAM
2. Ensure you have at least 4GB RAM available
3. Check disk space (need at least 500MB free)
4. Restart the application

**Problem**: Application crashes or freezes
**Solution**:
1. Check Windows Event Viewer for error details
2. Ensure system meets minimum requirements
3. Try running as administrator
4. Contact support if issues persist

## Uninstallation

### Standard Uninstallation
1. Open Windows Settings
2. Go to Apps & Features
3. Find "Enterprise Lead Management System"
4. Click "Uninstall"
5. Follow the uninstaller prompts

### Manual Cleanup (if needed)
If standard uninstallation fails:
1. Delete the application folder: `%LOCALAPPDATA%\Enterprise Lead Management System`
2. Remove desktop and Start Menu shortcuts manually
3. Clear any remaining registry entries (advanced users only)

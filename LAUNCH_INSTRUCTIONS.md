# Launch Instructions

## Quick Start

### Starting the Application
1. **Desktop Shortcut**: Double-click "Enterprise Lead Management System" on your desktop
2. **Start Menu**: Click Start → Search "Enterprise Lead Management System" → Click the application
3. **File Explorer**: Navigate to `%LOCALAPPDATA%\Enterprise Lead Management System` → Double-click the executable

### First Time Setup
1. The application will start automatically after installation
2. No additional setup or configuration required
3. The application runs completely offline

## Troubleshooting

**Solution**:

3. Check available disk space
4. Ensure sufficient RAM

**Problem**: Data not saving
**Solution**:

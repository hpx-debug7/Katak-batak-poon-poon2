V4U Lead Funnel CRM - Quick Reference Guide
Version 2.0.0

QUICK START
===========
1. Double-click desktop shortcut or Start Menu entry
2. Use default password: admin123 (change immediately)
3. Start adding leads and managing your sales pipeline

SYSTEM REQUIREMENTS
===================
- Windows 10 or later (64-bit)
- 4GB RAM minimum (8GB recommended)
- 500MB free disk space
- No internet required (fully offline)

DEFAULT CREDENTIALS
===================
Admin Password: admin123
Export Password: admin123
⚠️ CHANGE THESE IMMEDIATELY FOR SECURITY

MAIN FEATURES
=============
- Lead Management: Add, edit, search, filter leads
- Activity Tracking: Log interactions and follow-ups
- Export/Import: Excel file support with password protection
- Work Tracking: Monitor time and activities
- Follow-up System: Automated reminders and tracking

KEYBOARD SHORTCUTS
==================
Ctrl + N    - Add new lead
Ctrl + F    - Search/filter leads
Ctrl + E    - Export data
Ctrl + I    - Import data
F5          - Refresh data
Esc         - Close dialogs

NAVIGATION
==========
- Dashboard: Overview and quick access
- All Leads: Complete lead management
- Add Lead: Create new lead entries
- Due Today: Leads requiring attention
- Upcoming: Scheduled follow-ups
- Follow-up Mandate: Mandatory tracking
- Work Tracker: Time and activity monitoring

DATA MANAGEMENT
===============
- All data stored locally on your computer
- No cloud sync required
- Automatic backup system
- Export to Excel with password protection
- Import from Excel files

PERFORMANCE TIPS
================
- Close unused applications for better performance
- Restart application daily for optimal performance
- Use search and filters for large datasets
- Maintain at least 500MB free disk space
- Ensure 4GB+ RAM available

TROUBLESHOOTING
===============
Application won't start:
- Check system requirements
- Try running as administrator
- Check for antivirus interference

Slow performance:
- Close other applications
- Restart the application
- Check available disk space

Data not saving:
- Check disk space
- Verify write permissions
- Try running as administrator

INSTALLATION LOCATIONS
======================
Application: %LOCALAPPDATA%\Enterprise Lead Management System
Data: %LOCALAPPDATA%\Enterprise Lead Management System\data
Logs: %LOCALAPPDATA%\Enterprise Lead Management System\logs

UNINSTALLATION
==============
1. Windows Settings > Apps & Features
2. Find "Enterprise Lead Management System"
3. Click "Uninstall"
4. Follow prompts
